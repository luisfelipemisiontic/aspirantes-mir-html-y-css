Receta:Ensalada

--ingredientes--

los ingredientes necesarios para preparar la ensalada son:

  • pechuga
  • lechuga
  • tomate
  • cebolla
  • maiz

--preparación--

  1. cocinar pechuga
  2. picar cebolla
  3. picar tomate
  4. picar lechuga
  5. abrir lata de maiz
  6. mezclar todos los ingredientes
  7. servir inmediatamente

¡listo a disfrutar la comida!
